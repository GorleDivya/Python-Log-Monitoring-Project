# Python-Log-Monitoring-Project
Objective
Python-based log monitoring and analytics tool.
Requirements
1. Read logs from the file
2. Identify log levels: INFO, WARNING, ERROR
3. Provide analytics:
   - Total log count
   - Count per log level
   - Most frequent ERROR message
   - Filter logs by time range
4. Provide a CLI-based menu
5. Export analytics to a JSON report

Instructions
1.Ensure sample.log is present in the root directory.
2.Log file is parsed into structured objects
3.Results are exported to report.json.
4.Run the application : python main.py
