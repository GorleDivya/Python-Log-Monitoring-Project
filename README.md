# Python-Log-Monitoring-Project
Objective
A Python-based log monitoring and analytics tool.

Requirements
Read logs from the file
Identify log levels: INFO, WARNING, ERROR
Provide analytics:
Total log count
Count per log level
Most frequent ERROR message
Filter logs by time range
Provide a CLI-based menu
Export analytics to a JSON report
Python-based log monitoring and analytics tool.
- Read logs from file
- Identify INFO, WARNING, ERROR
- CLI menu-based analytics
- Filter logs by time range
- Export report to JSON
